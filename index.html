Step 1: HTML and CSS

This is the course that I made, and this is the website that I did just afterwards.

Apart from making use of the most common tools like flexbox and CSS grid, I also:

  • Designed the logo with Affinity Designer.
  • Adapted each image to be only double of the maximum size.
  • Made the website responsive.
  • Edited each picture to have a lighter or darker background to be more clearer.
  • Created all the content (images and texts).

I also made this website along with the course (content and instructions were provided though).

Step 2: GIT and GITHub

Being aware that I will be joining a team in the future together with the importance of the use of a VCS, I completed this course before doing anything else.

I also created the first repo of this folder and uploaded it (at this point in time) to GitHub to start practicing the main functions.

GIT is the most amazing technology I learned so far. Somehow HTML and CSS felt like an adaptation of what I already knew about Microsoft Office and Affinity Designer, into code. I cannot believe what GIT does. If I knew about it before...

Step 3: JavaScript

Having a long course ahead (67 hours in total), I decided to start using Codewars after learning the basics of JavaScript language. And... I got addicted! 100% recommended. Just check my profile to see all the katas that I completed. It really helps to remember most JavaScript concepts.

While being in the middle of the course, I took the decision to start a real project since I have the feeling that I am getting too far away from the main goal, which is being able to do websites. Also the amount of knowledge accumulated but yet not applied is huge.
