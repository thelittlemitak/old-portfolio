Hi! My name is Jose Riera Tur and I welcome you to my portfolio!

Here I show all the projects that I made so far and explain the steps that I followed for each one.

I also include the courses that I completed, together with the websites that I created to put every technology into practice.

This will also serve as a guideline for newcomers who are willing to start this amazing journey but don't know how. This methodology is a result of long research and good advice from professionals.

before the beginning...

Before I started this passion, I was already skilled in different areas (because of my education, career and hobbies) that have been of huge help during the journey. The most significant ones have been:

  • Game development (Unity and Godot engines and some basic C#)
  • Graphic design (Affinity Designer. I love it!)
  • Mathematics (They never left me: first at school; then as an audio technician)
  • Languages (I cannot imagine the learning curve without knowing EN or DE)
  • Good formal-writting practices (I wrote around 600 formal letters at my last job)
  • Microsoft Office (It was also very used at my last job)

And of course, the soft skills that got me this far without not giving up while being self-taught:

  • Fun (you will find why very soon)

let's go!...

Step 1: HTML and CSS

This is the course that I made, and this is the website that I did just afterwards.

Apart from making use of the most common tools like flexbox and CSS grid, I also:

  • Designed the logo with Affinity Designer.
  • Adapted each image to be only double of the maximum size.
  • Made the website responsive.
  • Edited each picture to have a lighter or darker background to be more clearer.
  • Created all the content (images and texts).

I also made this website along with the course (content and instructions were provided though).

Step 2: GIT and GITHub

Being aware that I will be joining a team in the future together with the importance of the use of a VCS, I completed this course before doing anything else.

I also created the first repo of this folder and uploaded it (at this point in time) to GitHub to start practicing the main functions.

GIT is the most amazing technology I learned so far. Somehow HTML and CSS felt like an adaptation of what I already knew about Microsoft Office and Affinity Designer, into code. I cannot believe what GIT does. If I knew about it before... (thinking now about all the times in my life that I created a copy (and a copy of a copy) of a file to create different branches).

Step 3: JavaScript

Having a long course ahead (67 hours in total), I decided to start using Codewars after learning the basics. And... I got addicted!

Being in the middle of the course, I took the decision to jump to React and start a real project (moremore.me) since I have the feeling that I am getting too away from the main goal, which is being able to do websites. Don't get me wrong: JS is important and I will finish the course and keep the codewars challenges, but I want to start applying what I know to real problems and start learning the chosen library.
